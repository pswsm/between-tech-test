## Description

A technical test for an API developer position!

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

To run the **mongo docker**, do:
```bash
$ docker compose up -d
```

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
